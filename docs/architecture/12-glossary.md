## 术语

- **源事实（Source of Truth）**：对某个主题给出权威陈述的长期文档。
- **持久规则（Durable Guidance）**：变化频率应当较低的稳定执行或协作规则。
- **工作流（Workflow）**：可复用的操作流程或过程指南。
- **临时变更规格（Temporary Spec）**：变更专用或阶段专用，不应演化为长期架构真相的文档。
- **历史遗留候选（Legacy Candidate）**：具有历史价值或与新结构重叠，后续可能被替代或改为指针的文档。
- **Patch**：本仓库处理和审核数据的最小单元。
- **`AGENTS.md`**：模块或项目的 agent 指南文件，应只承载持久操作规则，而不是完整业务真相。
- **标准 Skill 包**：位于 repo root `.agents/skills/<skill-name>/SKILL.md` 的可复用流程文档，只承载工作流，不承载完整业务真相。
- **arc42**：本项目用于组织项目级与模块级架构文档的结构化方法。
- **T05 Family**：legacy T05 拓扑模块与当前 T05-V2 模块的统称；两者在领域上连续，但仓库路径已分离。
